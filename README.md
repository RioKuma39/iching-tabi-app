# 易経を旅する Webアプリ

## 入口
- `index.html`：総合目次
- `quiz/index.html`：八卦学習クイズ
- `tarot/index.html`：イーチン旅タロット（一枚引き・三枚引き）
- `tarot/library.html`：64卦カード一覧・意味辞典
- `tarot/notes.html`：カード別ノート一覧・CSV書き出し

## カード画像
`tarot/cards/01.webp` 〜 `64.webp` の名前で配置してください。
現在同梱されている画像は01〜20です。21〜64も同じ場所へ追加すると、一覧ページとカード引きページの両方で自動表示されます。

## 64卦一覧ページ
- 「前の卦」「次の卦」で順番に閲覧
- 1〜64の番号ボタンから直接表示
- 番号入力で直接移動
- URLの `?no=10` のような指定で特定カードを直接表示
